# 说明书

说明书是对目录或文件的相关描述和说明，以便开发者更快更容易地了解当前目录或文件的作用和用法

## 名称

components

## 含义

全局组件目录

## 简介

此目录为项目的全局组件目录，请勿改动！

## 作用

提供全局组件的存放地址和自动导入

## 场景

全局组件

## 举例

请不要手动在此目录下创建全局组件文件！

1. 使用 `bash yipack new --comp=<组件名称>` 创建全局组件
2. 直接通过 `yi-<组件名称小写+短横线分隔符>` 引用全局组件

比如：

`yipack new --comp=c-1` 创建全局组件

`<yi-c1></yi-c1>` 使用全局组件
