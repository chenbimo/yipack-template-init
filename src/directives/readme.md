# 说明书

说明书是对目录或文件的相关描述和说明，以便开发者更快更容易地了解当前目录或文件的作用和用法

## 名称

apis

## 含义

全局指令目录

## 简介

此目录为项目的全局指令目录，请勿改动！

## 作用

提供全局指令的存放地址和自动导入

## 场景

全局指令

## 举例

请不要手动在此目录下创建全局指令文件！

1. 使用 `bash yipack new --directive=<指令名称>` 创建全局指令
2. 直接通过 `v-<指令名称>` 使用全局指令

比如：

1. `yipack new --directive=test-1` 创建全局指令
2. `<div v-test-1></div>` 使用全局指令
